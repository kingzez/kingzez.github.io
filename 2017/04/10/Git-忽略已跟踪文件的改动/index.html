Git忽略已跟踪文件的改动

4月 10, 2017 283

在实际使用中，有时候不可避免会有操作失误，将系统文件添加到版本库中，这时要怎么忽略已跟踪的文件？通过以下步骤：

  • 从 Git 的数据库中删除对于该文件的追踪；
  • 把对应的规则写入 .gitignore，让忽略真正生效；
  • commit＋push。

作用是将此文件移出版本控制，git不会记录此文件的修改，这样再将该文件加到.ignore中，与之对应的是
