CSS Dark Mode

Mar 11, 2020 423

简单实现，主要依靠 media query 的 prefers-color-scheme

@media screen and (prefers-color-scheme: light) {
  body {
    background-color: white;
    color: black;
  }
}

@media screen and (prefers-color-scheme: dark) {
  body {
    background-color: black;
    color: white;
  }
}

还可通过 css var 实现。
